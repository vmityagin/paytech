### Комбинированная pill-форма (инпут + кнопка)

Где: Баннерная секция "Если нужна помощь — перезвоним"
Пример: поле телефона + кнопка "Отправить заявку" в одном контейнере

Контейнер:
- высота: ~72px
- radius: ~36px (полный pill)
- padding: ~8px 8px 8px 24px (слева — под текст, справа — под кнопку)
- display: flex, align-items: center

Заливка контейнера:
- фон: rgba(255, 255, 255, .1) — очень слабый белый на тёмном фоне
- граница: 1px solid rgba(255, 255, 255, .3) — полупрозрачная белая
- тень: нет

Левая часть (поле):
- Лейбл: ~11-12px, regular(400), белый с прозрачностью (~rgba(255,255,255,.7))
- Значение / плейсхолдер: ~17-18px, regular(400), белый (~#FFFFFF)
- Фон поля: прозрачный
- Граница поля: нет

Правая часть (встроенная кнопка):
- Форма: pill, height ~52px, padding ~12px 28px, radius ~26px
- Фон: белый (~#FFFFFF)
- Текст: тёмно-зелёный (~#1A5D50), ~15px, medium(500)
- Тень: нет

Состояния:
- error лейбл: цвет → оранжевый/amber (~#FF8C00)
- error сообщение под контейнером: ~13px, оранжевый (~#FF8C00)
- focus контейнер: граница → rgba(255, 255, 255, .6) — ярче
- filled: значение белое, лейбл белый с прозрачностью

## Карточки

Где: Секция "Преимущества кредита на бизнес в Сбере" — 3 шт.
Пример: "Скидка до 5% на ставку", "Оформление онлайн до..."

Форма:
- padding: ~24-32px

- тень: subtle (~0 4px 12px rgba(0,0,0,.08))

Содержимое:
- 3D-иллюстрация в бирюзовых тонах (вверху)
- Заголовок ~18-20px bold

Расположение: сетка 3 колонки, отступы ~20-24px между карточками

## Резюме

- Элементов: 25 (14 уникальных типов)
- Кнопок: 4 варианта — primary dark pill, secondary outline pill, white pill, light pill
- Ссылок: 2 варианта — карточная зелёная, навигационная серая
- Карточек: 4 варианта — преимущества, калькулятор форма, калькулятор результат (инвертированная), поддержка
- Типографика: 6 уровней — H1 hero, H2 секций, H3 карточек, body, caption, акцентное число
- Шрифт: SB Sans (корпоративный шрифт Сбера), гротеск, визуально близок к Inter
- Общий стиль: корпоративный финтех в фирменной изумрудно-бирюзовой палитре Сбера, чистый и минималистичный с акцентом на 3D-визуал
- Характерные особенности: все кнопки pill-формы, 3D-иллюстрации во всех ключевых секциях, инвертированная карточка калькулятора, круглые кнопки "+" у аккордеонов вместо шевронов
